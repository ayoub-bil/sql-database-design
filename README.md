# Grocery Store Delivery System — Database Design

CS-A1150 Databases, Aalto University, Spring 2024

## Overview
Relational database for managing a grocery store home 
delivery system. Covers order management, product inventory, 
delivery routes, and collector scheduling.

## Key features
- 16 normalized tables in BCNF
- Foreign key constraints for data integrity
- 17 use case queries covering orders, pricing, inventory

## Tech
SQLite / SQLiteStudio

- `kauppa.sql` — full schema with sample data
